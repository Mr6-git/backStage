# 前言
首先安装nodejs git获取代码
# 安装运行
>npm install    安装依赖包 <br>
>npm start    安装完成运行项目

# 文件描述
* /theme.json		less样式变量<br>
* /src			源码路径<br>
* /src/componnet	自定义控件<br>
* /src/config		配置文件<br>
*     需要编辑文件，可以通过路径在 router.config.js里面寻找<br>
* /src/data		缓存数据<br>
* /src/enum		枚举字段<br>
* /src/net		网络请求相关<br>
* /src/page		模块文件<br>
* 	/page/application	主应用<br>
* 	/page/login			登录<br>
* 	/page/forgot		找回密码<br>
* /src/resource	资源文件如css image<br>
* /src/utils		常用函数<br>
